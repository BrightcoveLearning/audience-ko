    오디언스 모듈 시작하기

    이 항목에서는 동영상 참여도를 추적하는 데 사용되는 Audience 모듈의 개요를 제공합니다.
     

    Audience 모듈은 오디언스 지원 브라이트코브 플레이어 내부에서 보는 비디오에 대한 비디오 참여 데이터를 추적하는 데 사용됩니다. 보기 데이터는 비디오 클라우드에 저장되며 선택적으로 Eloqua, Marketo, HubSpot 및 Salesforce와 같은 인기 있는 마케팅 자동화 플랫폼에 동기화할 수 있습니다.

    Audience 모듈에 액세스하려면 Video Cloud에 로그인하고 탐색 헤더에서오디언스를클릭합니다. 시청자 홈페이지에는 동영상이 얼마나 오랫동안 시청되었는지, 시청한 동영상 및 시청자에 대한 통찰력을 제공하는 비디오 참여 데이터가 표시됩니다. 이를 통해 시청자의 참여도를 높이고 전환율을 높일 수 있습니다.

    홈페이지에서는 다음과 같은 기능을 제공합니다.

    비디오 성능 보기

    비디오 성능 섹션은 시청자 지원 플레이어 내부에서 재생된 비디오에 대해 높은 수준의 비디오 성능을 제공합니다.

    참고: Audience 분석 시스템은 일반적인 Video Cloud 분석 시스템과 별개이며 두 시스템은 데이터를 서로 다르게 추적합니다. 따라서 두 메트릭을 비교하면 일부 메트릭에서 차이가 나타납니다. 예를 들어, Video Cloud 분석 시스템은 비디오가 시작되자마자 조회수를 계산하는 반면 Audience 분석 시스템은 동영상에 10초까지 조회수를 계산하지 않습니다. 따라서 14명의 시청자가 동영상을 시작하지만 10초 전에 6명이 동영상을 중단하면 Video Cloud 분석은 14개의 조회수를 표시하고 Audience 모듈에는 8개가 표시됩니다.

    가장 많이 본 동영상

    [가장 많이 본 비디오] 차트는 선택한 기간 동안 시청자가 활성화된 플레이어에서 가장 많이 본 비디오를 표시합니다.

    가장 매력적인 동영상

    가장 매력적인 비디오 차트는 선택한 기간 동안 시청자가 활성화된 플레이어의 가장 매력적인 동영상을 표시합니다. 참여도는 평균 시청 비율입니다. 비디오 참여도를 계산하는 방법에 대한 자세한 내용은 비디오 참여도점수 계산 교육 비디오를참조하십시오.

    시청자 동향

    [뷰어 추세] 차트는 선택한 기간 동안의 시간에 따른 비디오 조회수를 표시합니다. 알려진 뷰어와 익명 뷰어의 뷰가 모두 표시됩니다. 그래프 위에 마우스를 놓으면 실제 데이터 포인트가 표시됩니다.

    참고: Marketo Munchkin 연결을 사용하는 경우 뷰어 추세 그래프에 알 수 없는 뷰어에 대한 데이터만 표시됩니다. Munchkin 연결을 사용할 때 사용자 세부 정보는 수집되지 않습니다. 자세한 사용자 정보를 보려면 Marketo REST 연결을 사용해야 합니다. 자세한 내용은REST API를 사용하여 Marketo와 비디오 클라우드 통합을참조하십시오.

    가망 고객 성과 조회

    가망 고객 정보를 보려면가망 고객탭을 누릅니다. Lead 탭은 계정에 오디언스 리드 양식이 생성된 경우에만 나타납니다. 잠재 고객 성과 섹션은 시청자 지원 플레이어 내부에서 재생된 비디오에 대해 높은 수준의 잠재 고객 성능을 제공합니다. 차트의기간을선택하여 시작합니다. 다음 옵션을 사용할 수 있습니다.

    • 7 데이즈
    • 30 데이즈

    청중 깔때기

    잠재고객 유입경로에는 선택한 기간 동안 잠재고객 잠재 고객 양식이 있는 시청자 지원 플레이어의 동영상에 대한 지표가 표시됩니다.

    • 동영상 노출수 - 로드된 동영상 수
    • 비디오 보기 - 비디오 재생이 시작되고 스트림이 시작될 때 기록된 횟수입니다 (되감기 또는 재생은 포함되지 않음). 개별 시청자의 척도가 아닙니다.
    • 가망 고객 화면 뷰 - 가망 고객 양식 뷰의 총 수
    • 전환 - 실행된 가망 고객 양식의 총 수

    상위 변환 비디오

    상위 변환 비디오 차트는 전환율이 가장 높은 비디오를 표시합니다. 전환율은 제출된 잠재 고객 양식 수를 비디오 노출수로 나눈 값입니다.

    참고: 구성할 수 있는 연결 수는 Video Cloud 구독에 따라 결정됩니다.

    오디언스 플레이어 만들기

    오디언스 플레이어옵션을 사용하면 오디언스 지원 브라이트코브 플레이어를 만들 수 있습니다. 비디오 참여 데이터는 시청자 지원 비디오 플레이어 내에서 비디오 콘텐츠를 볼 때만 기록됩니다. 오디언스가 활성화된 플레이어 생성에 대한 자세한 내용은오디언스가 활성화된 플레이어 생성을참조하십시오.

    가망 고객 양식 생성 및 관리

    가망 고객 양식옵션을 사용하면 데이터 캡처에 대한 가망 고객 양식을 생성할 수 있습니다. 리드 양식이 생성되면 오디언스가 활성화된 플레이어와 연결할 수 있습니다. 가망 고객 화면 생성에 대한 자세한 내용은오디언스 가망 고객 화면 생성을참조하십시오. 타사 마케팅 자동화 플랫폼 내에서 작성된 잠재 고객 양식을 만들고 사용할 수도 있습니다.

    전자 메일에 비디오 게시

    전자 메일로 게시옵션을 사용하면 플레이어, 비디오를 선택한 다음 HTML 내장 코드를 생성하여 비디오를 전자 메일로 보낼 수 있습니다. 자세한 내용은대상 모듈을 사용하여 전자 메일에 비디오 게시 를참조하십시오.

    참고: 미디어 모듈은 비디오를 전자 메일로 게시하는 기능도 제공합니다. 자세한 내용은전자 메일에 비디오 게시 를참조하십시오.

    비디오 활동 보기

    비디오 (비맵)옵션을 사용하면 비디오의 보기 활동을 볼 수 있습니다. 자세한 내용은비디오 활동 보기를참조하십시오.

    뷰어 프로파일 보기

    뷰어 프로필 (비맵)옵션을 사용하면 시청자별로 플레이어 보기 이벤트를 볼 수 있습니다. 이 옵션은 SSO 또는 사용자 지정 통합 또는 대상 추적을 사용할 때 활성화됩니다. 자세한 내용은뷰어 프로파일 정보 보기를참조하십시오.

    동기화 작업 보기

    [ 동기화 활동 ] 옵션은 오디언스에서 마케팅 자동화 플랫폼으로 동기화된 마지막 10개의 작업을 표시합니다. 동기화는 매시간 단위로 실행됩니다. 연결을 선택한 다음동기화 테스트단추를 클릭하여 동기화를 실행합니다. 이 기능은 이벤트 데이터가 마케팅 자동화 플랫폼으로 전송되는지 확인하기 위해 기다릴 필요가 없도록 새로운 통합을 테스트할 때 유용합니다. 테스트 동기화는 5분마다 한 번씩 시간당 최대 10회까지 실행할 수 있습니다.

    참고: 사용자 정의 통합을 사용하는 경우 동기화 작업이 생성되지 않습니다. 동기화 작업은 마케팅 자동화 플랫폼과의 통합을 사용하는 경우에만 생성됩니다.

    대상 설정 구성

    설정옵션을 사용하면 계정에 대한 몇 가지 기본 설정을 구성할 수 있습니다. 자세한 내용은사용자 설정 구성을 참고하십시오.

    대상 데이터 내보내기
