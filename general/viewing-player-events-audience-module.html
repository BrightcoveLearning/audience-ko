    대상 모듈에서 플레이어 이벤트 보기

    이 항목에서는 Audience 모듈에서 플레이어 이벤트를 보는 방법을 배우게 됩니다.

    Audience 모듈은 뷰 이벤트를 추적하는 기능을 제공합니다. 보기 이벤트가 Audience 모듈로 전송되는 빈도는 비디오 길이에 따라 동적으로 계산됩니다. 청중은 사분위당 하나씩 최소 4개의 이벤트를 보냅니다.

    참고: 뷰 이벤트 데이터는데이터 내보내기옵션을 사용하거나Audience API를사용하여 내보낼 수 있습니다.

    보기 이벤트를 보려면 대상 모듈을 열고 왼쪽 탐색에서 모든 최근 활동을클릭합니다. 이 페이지에는 지난 24시간 동안의 이벤트 활동이 표시됩니다.

    그만큼상호 작용 이벤트 표시링크가 활성화되는 경우사용자 지정 이벤트 또는 메트릭보기 이벤트에 첨부됩니다.

    페이지에 다음 정보가 표시됩니다.

    • 타임스탬프 - 비디오 보기의 날짜 및 시간
    • 비디오 이름 및 ID - 비디오 이름 및 비디오 클라우드 비디오 ID

    • 시청률 (% ) - 시청자가 비디오 시청을 중단한 지점을 비디오 길이로 나눈 값입니다. 예를 들어 시청자가 2분 비디오의 처음 5초를 본 다음 55초 지점으로 건너뛰고 5초 더 시청한 후 비디오를 중단하거나 페이지를 떠난 경우 시청률은 50% 로 기록됩니다.

      참고: 라이브 비디오의경우 오디언스에서 시청한 비율을 추적할 수 없습니다. 얼마나 오래 지속되는지 알 수 없기 때문에 Audience 모듈은 시청한 총 시간만 추적합니다.
    • 시청 시간 - 총 시청 시간
    • 외부 ID - 뷰어를 식별하는 데 사용되는 값입니다 (예: Eloqua GUID, 마케토 GUID, 허브 스팟 GUID, SSO GUID 또는 뷰어를 식별하는 데 사용된 다른 고유 값).
    • 상태 - 이벤트 상태
      • 처리되지 않음 - 이벤트가 마케팅 자동화 플랫폼에 동기화되지 않았습니다.
      • 동기화 - 이벤트가 마케팅 자동화 플랫폼에 성공적으로 동기화되었습니다.
      • sync_disabled - 이벤트를 건너뛰도록 동기화가 비활성화된 상태로 대상 모듈이 구성되었습니다.
      • 실패 - 오류 동기화. 도구 설명에 발생한 오류에 대한 세부 정보가 제공됩니다.
      • unknown_contact - 알 수 없는 사용자이므로 이벤트가 동기화되지 않았습니다.
      • 해당사항 없음 - 해당 사항 없음, 사용자 지정, 대상 추적 또는 Marketo Munchkin 통합을 사용할 때 표시됨

    이벤트 데이터는 페이지 상단의 드롭다운 목록을 사용하여 상태별로 필터링할 수도 있습니다.

    상호 작용 이벤트 보기

    사용자 지정 메트릭 또는 이벤트가 이벤트 보기에 연결된 경우상호 작용 이벤트 표시링크가 활성화됩니다. 예를 들어 오디언스 지원 플레이어에서 갤러리 인페이지 환경을 사용하고 있고 해당 환경이링크, 카드 또는 기타 유형의 인터랙티브 요소로구성되어 있는 경우 경험은 노출 및 전환 이벤트를 Audience 모듈로 보냅니다. 이러한 이벤트는 대상 사용자 정의 필드를 사용하여 오디언스로 전달됩니다. 이벤트 데이터는모든 최근 활동의일부로 대상 모듈에 나타납니다.

    참고: Audience 사용자 지정 지표 및 이벤트를 사용할 때도 상호 작용 이벤트가나타날 수 있습니다.

    다음 데이터가 전달 될 수 있습니다.

    • 이벤트 1 - 뷰어가 상호 작용할 기회가 있었던 CTA 이름의 쉼표로 구분된 목록 (노출)
    • 이벤트 2 - 뷰어가 상호작용한 CTA 이름의 쉼표로 구분된 목록 (변환)
    • 지표 1 - 총 CTA 노출 수
    • 지표 2 - CTA 전환의 총 수 (예: 클릭 링크, 양식 제출 등)

    이벤트를 보려면 헤더에서상호 작용 이벤트 표시링크를 클릭합니다.

    기타 데이터 보기 옵션

    대상 데이터는 다른 방법으로 보기/내보낼 수 있습니다.

    • 비디오 활동 - 비디오별로 시청자 활동을 보는 데 사용됩니다. 자세한 내용은비디오 활동 보기를참조하십시오.
    • 뷰어 프로필 - 시청자별로 비디오 활동을 보는 데 사용됩니다. 자세한 내용은뷰어 프로파일 정보 보기를참조하십시오.
    • 데이터 내보내기 - 비디오 보기 및 리드 데이터를 내보내는 데 사용됩니다. 자세한 내용은대상 모듈에서 사용자 데이터 내보내기 를참고하십시오.
    • 오디언스 API - 오디언스 API는 프로그래밍 방식으로 동영상 보기 및 리드 데이터를 검색하는 데 사용할 수 있습니다. 자세한 내용은개요: 오디언스 API
