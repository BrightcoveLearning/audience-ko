    대상 모듈을 사용하여 시청자 추적

    이 항목에서는 Audience 모듈을 사용하여 시청자 활동을 추적하는 방법을 배우게 됩니다.

    Audience 모듈은 보기 활동을 추적할 수 있는 기능을 제공합니다. 이 시나리오에서는 데이터 보기 마케팅 자동화 플랫폼에 전송 되지 않습니다. 이에 대한 일반적인 사용 사례 중 하나는 내부 교육 비디오입니다. 대상 추적은 내부 직원의 조회 활동을 추적하는 데 사용할 수 있습니다. 대상 추적을 사용하려면 다음 단계가 필요합니다.

    1. 대상 추적 설정 구성
    2. 잠재고객 추적을 위한 플레이어 및 리드 양식 구성
    3. 비디오 데이터 보기

    대상 추적 설정 구성

    대상 추적 설정을 구성 하려면 다음과이 같이 하십시오.

    1. 대상 모듈을 열고 왼쪽 탐색에서데이터 연결을 클릭합니다.
    2. 대상 추적섹션을 찾아보기를클릭합니다.
    3. 기본적으로 대상 추적은 모든 계정에 대해 사용하도록 설정됩니다. 필요에 따라 설정을 편집합니다.
      • 연결 이름 - 연결을 식별합니다.
      • 연결 상태 - Audience 모듈에서 비디오 클라우드 비디오 참여 데이터를 저장할지 여부를 결정합니다.
        • 사용 - 데이터가 수집됩니다.
        • 사용 안 함 - 데이터가 수집되지 않습니다 (연결이 비활성화됨).
      • 알 수 없는 뷰어에 대한 데이터 추적 - 알 수 없는 뷰어에 대해 추적 데이터를 기록할지 여부를 결정합니다. 오디언스 추적을 사용할 때 시청자를 식별하기 위해오디언스 리드 양식을사용할 수 있습니다.
        참고: 잠재고객 추적을 사용하면 24시간 동안 최대 1,000명의 시청자가 추적됩니다.
      • 브라이트코브는 데이터를 얼마나 오래 보존해야 합니까? - 데이터 보존 정책에 따라 시청 데이터와 잠재 고객 양식 데이터 (Audience 리드 양식을 사용하는 경우) 가 Video Cloud 계정에 보존되는 기간이 결정됩니다. 개인 정보 보호 정책에 따라 보존 설정을 조정해야 합니다. Video Cloud에 사용자 데이터를 보관할 필요는 없습니다.

    오디언스 추적을 위한 플레이어 및 리드 양식 구성

    시청 활동은 시청자 지원 비디오 플레이어 내에서 비디오를 볼 때만 추적할 수 있습니다. 오디언스 추적을 사용할 때는 오디언스 지원 플레이어를 사용해야 합니다. 비디오 재생 중에 리드 양식을 표시하여 시청자 정보를 캡처할 수 있습니다. 잠재 고객 추적을 사용할 때는 잠재 고객 양식이 필요하지 않습니다.

    오디언스 지원 플레이어 및 리드 양식으로 오디언스 추적을 구성하려면 다음 단계를 따르십시오.

    1. 왼쪽 탐색에서대상 플레이어를클릭합니다.
    2. 사용할 대상 플레이어를 결정합니다. 시청자가 활성화된 플레이어가 없으면플레이어 만들기를 클릭하여 플레이어를만듭니다. 오디언스가 활성화된 플레이어 만들기에 대한 자세한 내용은오디언스가 활성화된 플레이어 만들기를참조하십시오.
    3. 리드 양식은 필수는 아니지만 잠재 고객 양식을 사용하여 시청자에 대한 정보를 수집하고 익명 시청자를 알려진 뷰어로 전환할 수 있습니다. 가망 고객 양식을클릭하고 가망 고객 양식을 생성합니다. 가망 고객 화면 생성에 대한 자세한 내용은오디언스 가망 고객 화면 생성을참조하십시오.
    4. 오디언스 플레이어를클릭합니다.
    5. 사용할 오디언스가 활성화된 플레이어를 찾습니다. 플레이어의연결대상 값을대상 추적으로설정합니다.
    6. 가망 고객 양식을 사용하는 경우 가망 고객 양식 드롭다운 목록에서가망 고객 양식을선택합니다.

    비디오 데이터 보기

    잠재고객 추적을 사용할 때 비디오 데이터를 보는 데 사용할 수 있는 몇 가지 옵션이 있습니다.

    • 시청자 활동 (이벤트) - 대상 모듈의 일부로 제공되는 이 옵션을 사용하여 Audience 모듈 내의 비디오 데이터를 볼 수 있습니다. 자세한 내용은대상 모듈에서 플레이어 이벤트 보기를참조하십시오.
    • 시청자 프로필 - Audience 모듈의 일부로 제공되는 이 옵션을 사용하여 Audience 모듈 내의 사용자 프로필 값을 기반으로 비디오 데이터를 볼 수 있습니다. 자세한 내용은뷰어 프로파일 정보 보기를참조하십시오.
    • 데이터 내보내기 - Audience 모듈의 일부로 제공되는 이 옵션을 사용하여 비디오 보기 및 리드 데이터를 내보낼 수 있습니다. 자세한 내용은대상 모듈에서 사용자 데이터 내보내기 를참고하십시오.
    • 오디언스 API - 오디언스 API는 프로그래밍 방식으로 동영상 보기 및 리드 데이터를 검색하는 데 사용할 수 있습니다. 자세한 내용은개요: 오디언스 API
